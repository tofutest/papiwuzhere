Papi Site — Content Guide

How to add new entries to each section of the site.

📓 Field Journal (Blog Posts)

These are the main entries on the home page — Papi's territorial accomplishments.

File location

Create a new .md file in the _posts/ folder. Name it:

YYYY-MM-DD-short-title.md

Example: _posts/2026-06-10-fire-hydrant-on-leary.md

What to put in the file

---
layout: post
title: "Your Entry Title Here"
date: 2026-06-10
time: "7:32 AM"
location: Ballard
icon: 🚒
category: Infrastructure
image: /your-image.png
image_alt: Description of image
targets:
  - "🚒 fire hydrant"
  - "🌊 puddle nearby"
rating: 5
quote: "A classic. I will return."
---

Your entry text goes here. Write as much as you want.
This is what shows on the full entry page.

What I need from you

  • Required: Title, date, the text
  • Required: Location, time
  • Optional: Image file (drop it in the project folder), icon emoji, targets, rating (1–5), closing quote
The first sentence or two automatically becomes the preview on the home page. The rest shows when the reader clicks "Read full entry."

🎨 Art Gallery

Individual artworks that appear on the Papi Art page.

File location

Create a new .md file in the _artworks/ folder. Name it anything descriptive:

_artworks/starry-night.md

What to put in the file

---
layout: artwork
title: "The Starry Night (After van Gogh)"
badge: "🖼️ Masters Series"
meta: "Mixed media on canvas · Ballard · Summer 2026"
image: /starrynight.jpg
image_alt: Papi's take on Starry Night
preview: "One sentence or two for the gallery index. This is what visitors see before clicking through."
quote: "The swirls spoke to me. I responded in kind."
provenance: "<strong>Medium:</strong> Instinct &nbsp;·&nbsp; <strong>Duration:</strong> 3 seconds"
---

Full description of the piece. This only shows on the individual artwork page.
As many paragraphs as you want.

What I need from you

  • Required: Title, one-line preview, full description text
  • Required: Image file
  • Optional: Badge label, meta line, closing quote, provenance details
The preview text shows on the gallery index. Everything else (image, full text, buy button, quote) shows on the individual artwork page.

💌 Fan Mail

Entries on the Fan Mail page, shown newest first, all in full — no click-through needed.

File location

Create a new .md file in the _fanmail/ folder. Name it with the date:

_fanmail/2026-06-10-response.md

What to put in the file

---
date: 2026-06-10
image: /your-image.jpg
image_alt: Description of image
---

The text that goes with this entry.
Can be a sentence or several paragraphs.

What I need from you

  • Required: Date, the text
  • Optional: Image file
Entries stack newest-first. Just drop a new file in _fanmail/ with today's date and it will appear at the top automatically.

🖼️ Adding Images

Drop image files directly into the main project folder (PapiWuzHere/). Then reference them in your entry with a leading slash:

image: /myimage.jpg

Supported formats: .jpg, .png, .avif

After adding or editing files, commit and push to the jekyll-blog branch to publish to the live site.
